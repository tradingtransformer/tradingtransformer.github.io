Infinity Memory：何も忘れない、ひとつのオーケストレーター

NoArk は中心に立つ、プライバシー優先のシステムオーケストレーターです——Siri がそうあるべきだった、能動的で文脈を理解する Infinity Memoryとして——何も忘れず、あなたの文脈・会話・知識は限界に突き当たることなく引き継がれ、あなたのすべてのデバイスにまたがって働きます。

他のAIはやり方を教える。NoArkは何をすべきかを教える。

ほとんどのAIアシスタントは、あなたが何を望むかをすでに決めた後でタスクを実行するために作られています——コードを書く、メールを下書きする、ページを要約する。それがやり方です。NoArkは、その前に来るもっと難しい問いのために作られています：いま本当に取り組む価値があるのは何か、そしてなぜか？

行き詰まったとき、必要なのはもう一つの答えではありません——どの一手が重要かを知ることです。NoArkは、ニュースがあなたにとって何を意味するかをワンタップで分析するライブの世界のヘッドライン、Fiat Luxタブでの歴史上もっとも賢い人々のAIによる助言、そして市場・仕事・暮らしにわたる40以上の優先順位を見つけるワークフローをもたらします。単なる答えではなく導き——あなた自身のために、何に取り組む価値があるかを教えます。

実行するアシスタントと競うのではなく、その一段上に位置します。NoArkは内部でClaudeのFableモデルやその他のフロンティアモデルにも振り分けるので、やり方の上に立って、あなたが何をを正しく選べるよう手助けします。

名前に込めた物語

NoArk はノアの方舟（Noah's Ark）に由来します。言い伝えによれば、ノアは120年もの歳月をかけて方舟を造りました——そしてその大半の間、海から遠く離れた陸地で巨大な船を造る彼を、隣人たちは嘲笑っていました。それでも彼は造り続けました。周囲の誰も本気にしなかった大洪水が来ることを、彼には見えていたからです。

いま訪れつつあるエージェントAIの時代についても、私たちは同じように感じています。それがどれほど徹底的に、そしてどれほど速くすべてを変えるか、多くの人はまだ備えができていません。NoArk は私たちの方舟です——洪水に先んじて、辛抱強く造り上げ、あなたにとって最も大切なもの（知識・判断・記憶）を安全に未来へ運びます。

そしてもう一つの読み方があります——「no ark（方舟はない）」。この急速に変わるエージェントの時代に、出来合いの方舟であなたを救いに来る人はいません。あなた自身が方舟にならなければならないのです。NoArk のAIエージェントは、まさにそれを助けるために造られています。

「1984年、クロード・シャノンは、人工知能が2001年までに達成すべき4つの目標を掲げた。世界チャンピオンに輝くチェスプログラム、『ニューヨーカー』誌に作品が採用される詩のプログラム、難解なリーマン予想を証明する数学プログラム、そして『最も重要なもの』として、プライムレートを50パーセント上回る銘柄選定プログラムである。『これらの目標は』と彼は半ば冗談めかして言った、『愚かで、エントロピーを増大させ、好戦的な人類が、より論理的で、省エネで、友好的な種——コンピュータ——に取って代わられる段階の始まりを告げるかもしれない』。」

バイブトレーディングとは何ですか？

バイブトレーディングとは、人がすべてのチャートを見張るのではなく、AIエージェントに市場全体の「空気感」——センチメント、ニュースの流れ、株式や予測市場のリアルタイムなシグナル——を読み取らせ、それに基づいて行動させることです。NoArkのエージェントは新しいデータから継続的に学習するため、戦略はリアルタイムに適応し、人は時間とともに後退していきます。これはバイブコーディング（vibe coding）のトレーディング版です。ソフトウェアエンジニアが時間とともにAIエージェントの書いたコードをますます受け入れていくのと同じように、投資家もAI主導の取引をますます信頼するようになります——モデルが実力を証明し続けるにつれ、人による監督は後退していきます。

私たちのアプローチは、世界の市場——米国、香港、上海を含む——の上場企業への投資に注力し、レバレッジを使わずロングオンリーのポジションを維持します。低頻度トレーディングの会社として、持続可能な成長と慎重なリスク管理を重んじる、規律ある研究主導の戦略を強調します。

「価格チャートに『ヘッド・アンド・ショルダーズ』や『急落するネックライン』といったものを使ってあれほど熱心に取り組むテクニシャンたちは、私に言わせれば、重要なデータの非常にノイズの多い再現を相手にしているのだと思う。」

— クロード・シャノン

私たちはその見方に賛同します——だからこそ、チャートのパターンを追いかけるのではなく、ファンダメンタルズと、規律ある低頻度の戦略に注力するのです。

NoArk は Apple Intelligence と何が違いますか？

最も重要な二つの点で違います。提供状況：Apple Intelligence は現実の規制上の摩擦に直面してきました——その展開は欧州連合（デジタル市場法のもと）でも中国本土でも遅延・制限され、米国のように自由に使える状態になったことはありません。プライバシー：Apple Intelligence はパーソナル・コンテキストを構築して本当に役立つために、あなたのデバイス全体——アプリ、画面に表示されている内容、写真ライブラリのすべて——にまで手を伸ばします。自分で完全には制御できないシステムに与えるには、あまりに多くの可視性です。

NoArk は正反対の道を行きます。Siri がそうあるべきだった、能動的で文脈を理解する Infinity Memory として設計されており——既定でプライバシー優先です。あなたの記憶はデバイスとあなた自身の iCloud にローカルに保存され、私たちのサーバーに蓄積されることはなく、あなたの注意もデータも決して収益化しません。それでもなお、すべてのデバイスにまたがる、文脈を理解したシステム全体のオーケストレーターが手に入ります——デジタル人生のすべてを差し出すことなく。

バイブトレーディングとは、人がすべてのチャートを見張るのではなく、AIエージェントに市場全体の「空気感」——センチメント、ニュースの流れ、株式や予測市場のリアルタイムなシグナル——を読み取らせ、それに基づいて行動させることです。NoArkのエージェントは新しいデータから継続的に学習するため、戦略はリアルタイムに適応し、人による監督は時間とともに後退していきます。

AIエージェントは予測市場に役立ちますか？

役立ちます——ただし、優れたエージェントは「やってみなさい」と背中を押すのではなく、率直な判断を返します。以下は、Charlie（チャーリー・マンガーをモデルにした当社のAIエージェント）に「どうすれば若者に予測市場を使わせられるか」と尋ねたときの、飾らない答えです：

核心の結論：私は若者に予測市場を勧めません。むしろ使うなと言います。

理由は三つ：

  1. 若者に最も足りないのは情報ではなく、元手だ。予測市場では上位1%のユーザーが利益の大半を取り、残り99%の長期的な期待リターンはゼロに近づく。数千ドルを手にプロと張り合うのは投資ではなく、お金を差し出す行為だ。
  2. 予測市場は短い反応ループを鍛えるが、若者が最も育てるべきは遅延満足の力だ。数分や数時間で結果が分かるフィードバックは、本質的にスロットマシンと変わらない。若者に最も必要なのは「今日正しいことを一つやり、十年後に結果を見る」という習慣——それこそが複利の力だ。
  3. 予測市場はあなたを賢く感じさせるが、豊かにはしてくれない。10個の出来事を当てても、元手は増えず、技能も増えず、思考の枠組みも変わらない。本当の富は価値を生み出すことから来るのであって、予測を当てることからは来ない。

それでも若者が使うなら：厳しい上限を設け（月500ドルを超えない）、情報源としてだけ使い（オッズは見ても賭けはしない）、自分の独立した判断の代わりには決してせず、浮いた時間とお金は読書と思考に回すこと。

まとめ：若者の最も貴重な資産は時間と可塑性だ。予測市場は時間を細切れにし、注意を短期化し、思考をギャンブル化する——そのどれもが、彼らが最も育てるべき能力を蝕む。本当に賢い若者は、良書を読み、複利を生むスキルを学び、価値を生むシステムを築くことに力を注ぐ。

予測市場はあなたを賢くしない。ただ賢くなった気にさせるだけだ。そして賢くなった気でいることこそ、愚かさへの最短の道だ。

——Charlie

TradingViewについてどう思いますか？

チャートは重要ではありません。情報理論の父であり、それ自体が傑出した投資家でもあったクロード・シャノンはこう述べています。「価格チャートに『ヘッド・アンド・ショルダーズ』や『急落するネックライン』といったものを使ってあれほど熱心に取り組むテクニシャンたちは、私に言わせれば、重要なデータの非常にノイズの多い再現を相手にしているのだと思う」。TradingViewやそれに類するツールは、それ自体としては問題ありませんが、価格チャートは企業を実際に動かすものの、ノイズだらけの反映にすぎません。私たちはその注意を、チャートのパターンを追いかけることではなく、ファンダメンタルズと規律ある低頻度の戦略に向けたいと考えています。

NoArk は Apple Intelligence、Google の Gemini、Manus と何が違いますか？

NoArk は、もう一つの何でも屋のチャットボットを目指してはいません。私たちは、従来であればフルタイムのエグゼクティブ・アシスタントを雇うようなハイエンドのお客様のために、最も信頼でき、最も客観的な AI エージェントを構築しています。Apple Intelligence、Google の Gemini、Manus はいずれも優れた汎用ツールですが、最終的にはプラットフォームやエコシステム、そして広告やデータに基づくビジネスモデルに仕えています。NoArk が仕えるのは、あなただけです。私たちはあなたの注意も、あなたのデータも収益化しません。だからこそ Charlie は、あなたの市場、意思決定、日々の雑務について、利害のしがらみのない率直な判断を示せるのです——一流のエグゼクティブ・アシスタントに期待するような慎み深さと信頼性を、一つのアプリで。

NoArk は私のデータをどう扱いますか？

あなたのデータは、あなたのものであり続けます。私たち自身があなたのデータを収集することはありません。NoArk で作成したものはすべて、あなたのデバイスとあなた自身の iCloud にローカルで保存され——私たちのサーバーには保存されません。Charlie が何かを考える必要があるとき、NoArk はクラウド上の大規模言語モデルと API を介してやり取りします。そして、それらの提供元の API 利用規約のもとでは、あなたのコンテンツは保存されることも、モデルの学習に使われることもありません。要するに、あなたの記憶はあなたとともにあり、私たちがそれを溜め込むことは決してなく、私たちが呼び出す AI モデルもそれを保持しないのです。そしてもう一つ理由があります。データを溜め込まないことで、上昇し続けるストレージのコストも抑えられるのです——一石二鳥、あなたにとっても、私たちにとっても良いことです。

NoArk のメッセージはエンドツーエンドで暗号化されていますか？Apple や NoArk は私のメッセージを読めますか？

はい——NoArk はエンドツーエンド暗号化メッセージに対応しました。あなたと、あなたがメッセージを送る相手以外の誰も読めないように設計されています。メッセージを送信すると、NoArk はそれをあなたの端末上で暗号化された封筒に封入し、その判読不能な暗号文だけを CloudKit——Apple の公開データベース——にアップロードします。CloudKit は純粋にゼロ知識のリレーとして機能するだけです。

復号できる内容とすべての鍵は、あなたの端末から決して出ません。Apple が保持するのは読めないスクランブル化された暗号文だけで、しかも一時的に保存されるにすぎず、送信者が 7 日間の TTL の後にそれを削除します。鍵を持つ2台の端末だけが、実際にメッセージを復号して読むことができます——Apple にも、私たちにも読めません。

なぜ NoArk という名前なのですか？

NoArkには二重の意味があります。第一に、ノアの方舟（Noah's Ark）——変化の洪水を、大切なものを乗せて安全に渡るための船——を想起させます。言い伝えによれば、ノアは120年もの歳月をかけて方舟を造り、その大半の間、陸地で巨大な船を造る彼を隣人たちは嘲笑っていました——それでも彼は造り続けました。ほかの誰も本気にしなかった大洪水が来ることを、彼には見えていたからです。いま訪れつつあるエージェントAIの時代についても、私たちは同じように感じています。第二に、「no ark（方舟はない）」と読むこともできます。急速に変わるエージェントの時代に、出来合いの方舟であなたを救いに来る人はいません。あなた自身が方舟にならなければならない——NoArkのAIエージェントは、あなたの知識・判断・記憶を前へ運ぶことで、まさにそれを助けます。

TradingTransformer という名前はどこから来たのですか？

TradingTransformer は Transformer——Google の2017年の論文「Attention Is All You Need」で発表されたニューラルネットワークのアーキテクチャ——にちなんで名づけられました。私たちはその論文がどれほど重要になるかを早くから見抜いており、私たち自身のパーソナライズド Transformer の研究は当時、最優秀論文賞のノミネート（best-paper nominee）を受けました。TradingTransformer はその研究——Transformer モデルを市場に応用する取り組み——から生まれ、NoArk はその次の一歩です。すなわち、同じ「注意（attention）」を汎用 AI エージェントのパーソナライゼーションとプライバシーに向けること。知能があなたのために働き、あなたを覚え、そしてあなたのデータをすべてあなた自身のデバイスに保つようにします。

シャノンのエントロピー（情報量）とは何ですか？

1948年、クロード・シャノンはエントロピーを、情報の流れが持つ不確実性——いわば「驚き」の度合い——として定義しました。メッセージが予測しにくいほど、それが運ぶ情報は多く、「ビット」で測られます。これはデータ圧縮から誤りのない通信まで、デジタル時代の数学的な土台となりました。投資家にとっても直感的です。すでに完全に予測できる価格は何の情報も優位ももたらしません——本当のシグナルは、市場がまだ織り込んでいない部分に宿ります。NoArkのエージェントは、誰もがすでに見えているノイズではなく、その「残された不確実性」を見つけて活かすために作られています。

マクスウェルの悪魔とシャノンのデーモンにはどんな関係がありますか？

どちらも、ランダムさから秩序を生み出すように見える「魔物」です。マクスウェルの悪魔は、物理学者ジェームズ・クラーク・マクスウェルが1867年に思い描いた仮想の存在で、速い分子と遅い分子を選り分けて気体のエントロピーを下げ——熱力学第二法則を破るかに見えます。この逆説が解かれたのは20世紀になってからでした。レオ・シラードが、そして後にロルフ・ランダウアーが、悪魔は選り分けのために情報を取得し最終的に消去しなければならず、その情報の消去自体がエネルギーを要することを示したのです。情報と物理的エントロピーは同じコインの裏表だった——まさにシャノンの情報理論が精密に捉えた結びつきです。彼にちなんで名づけられた「シャノンのデーモン」、すなわちボラティリティを収穫するリバランス戦略は、その金融版のこだまです。変動する価格の無秩序でランダムな揺れから「秩序」——着実な複利——を取り出すのです。両者の深い教訓は同じです。秩序はタダでは手に入りません。マクスウェルの悪魔はエネルギーで支払い、シャノンのデーモンは取引コストで支払う。だからこそNoArkは、秩序を収穫するコストが収穫する秩序を決して上回らない、規律ある低頻度の戦略を好むのです。

「シャノンのデーモン」とは何で、NoArkはそれを使いますか？

テクニカル分析への懐疑に加え、クロード・シャノンは現実の市場の実務的な摩擦も認識していました。彼は、ボラティリティ駆動の高度に体系的なポートフォリオ・リバランス戦略——時に「シャノンのデーモン」と呼ばれる——を概念的に発展させましたが、その数学的アイデアの多くをウォール街で実行することはありませんでした。頻繁な取引の高い手数料と取引コストが、単純なバイ・アンド・ホールド戦略に対する数学的優位をたちまち食いつぶす、と結論づけたからです。NoArkも同じ教訓を胸に刻み、コストが優位を侵食しない、規律ある低頻度の戦略を好みます。
