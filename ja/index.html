私たちの創業の着想：AIに対するクロード・シャノンのビジョン

1984年、情報理論の先駆者クロード・シャノンは、AIの銘柄選定プログラムが汎用人工知能（AGI）の「最も重要な」マイルストーンになると予言しました。このビジョンこそ、私たちが人々のために働き——そして投資する——AIエージェントを作る理由です。

「1984年、クロード・シャノンは、人工知能が2001年までに達成すべき4つの目標を掲げた。世界チャンピオンに輝くチェスプログラム、『ニューヨーカー』誌に作品が採用される詩のプログラム、難解なリーマン予想を証明する数学プログラム、そして『最も重要なもの』として、プライムレートを50パーセント上回る銘柄選定プログラムである。『これらの目標は』と彼は半ば冗談めかして言った、『愚かで、エントロピーを増大させ、好戦的な人類が、より論理的で、省エネで、友好的な種——コンピュータ——に取って代わられる段階の始まりを告げるかもしれない』。」

出典：A Mind at Play、AIの先駆者クロード・シャノンの2017年の伝記。彼は1984年にこれを書き、2001年に亡くなりました。

バイブトレーディングとは何ですか？

バイブトレーディングとは、人がすべてのチャートを見張るのではなく、AIエージェントに市場全体の「空気感」——センチメント、ニュースの流れ、株式や予測市場のリアルタイムなシグナル——を読み取らせ、それに基づいて行動させることです。NoArkのエージェントは新しいデータから継続的に学習するため、戦略はリアルタイムに適応し、人は時間とともに後退していきます。これはバイブコーディング（vibe coding）のトレーディング版です。ソフトウェアエンジニアが時間とともにAIエージェントの書いたコードをますます受け入れていくのと同じように、投資家もAI主導の取引をますます信頼するようになります——モデルが実力を証明し続けるにつれ、人による監督は後退していきます。

私たちのアプローチは、世界の市場——米国、香港、上海を含む——の上場企業への投資に注力し、レバレッジを使わずロングオンリーのポジションを維持します。低頻度トレーディングの会社として、持続可能な成長と慎重なリスク管理を重んじる、規律ある研究主導の戦略を強調します。

「価格チャートに『ヘッド・アンド・ショルダーズ』や『急落するネックライン』といったものを使ってあれほど熱心に取り組むテクニシャンたちは、私に言わせれば、重要なデータの非常にノイズの多い再現を相手にしているのだと思う。」

— クロード・シャノン

私たちはその見方に賛同します——だからこそ、チャートのパターンを追いかけるのではなく、ファンダメンタルズと、規律ある低頻度の戦略に注力するのです。

出典：Concentrated Investing（2016年）。

バイブトレーディングとは何ですか？

バイブトレーディングとは、人がすべてのチャートを見張るのではなく、AIエージェントに市場全体の「空気感」——センチメント、ニュースの流れ、株式や予測市場のリアルタイムなシグナル——を読み取らせ、それに基づいて行動させることです。NoArkのエージェントは新しいデータから継続的に学習するため、戦略はリアルタイムに適応し、人による監督は時間とともに後退していきます。

AIエージェントは予測市場に役立ちますか？

はい。NoArkのAIエージェントは予測市場のオッズとシグナルを読み取り、それらをAIが生成した明快なインサイトに変換して、より的確な判断を助けます。NoArkがあなたの代わりに予測市場で取引を行うことはありません——すべての判断はあなたが握っています。

クロード・シャノンはAIと銘柄選定について何を予言しましたか？

1984年、クロード・シャノン——情報理論の父——は、市場を上回るAIの銘柄選定プログラムを、チェス、詩、リーマン予想の証明と並ぶ汎用人工知能（AGI）の「最も重要な」目標として挙げました。この予言こそ、TradingTransformerとNoArkの創業の着想です。

NoArk — Infinity Memory とは何で、無料ですか？

NoArk — Infinity Memory は、バイブトレーディングと予測市場インサイトのために作られた、手頃な汎用AIエージェントアプリです。市場・生産性・旅行・生活管理にわたる40以上のプリセットワークフローで、テキスト・音声・写真・ファイルを使ってCharlieと対話できます。iPhone・iPad・Mac（Apple Silicon）・Apple Vision で無料ダウンロードできます。アプリには Free・Silver・Gold・Diamond の4段階があります。有料プランは 月額9.99ドルから——早割として通常価格の50%オフです。

NoArk は Apple Intelligence、Google の Gemini、Manus と何が違いますか？

NoArk は、もう一つの何でも屋のチャットボットを目指してはいません。私たちは、従来であればフルタイムのエグゼクティブ・アシスタントを雇うようなハイエンドのお客様のために、最も信頼でき、最も客観的な AI エージェントを構築しています。Apple Intelligence、Google の Gemini、Manus はいずれも優れた汎用ツールですが、最終的にはプラットフォームやエコシステム、そして広告やデータに基づくビジネスモデルに仕えています。NoArk が仕えるのは、あなただけです。私たちはあなたの注意も、あなたのデータも収益化しません。だからこそ Charlie は、あなたの市場、意思決定、日々の雑務について、利害のしがらみのない率直な判断を示せるのです——一流のエグゼクティブ・アシスタントに期待するような慎み深さと信頼性を、一つのアプリで。

NoArk は私のデータをどう扱いますか？

あなたのデータは、あなたのものであり続けます。私たち自身があなたのデータを収集することはありません。NoArk で作成したものはすべて、あなたのデバイスとあなた自身の iCloud にローカルで保存され——私たちのサーバーには保存されません。Charlie が何かを考える必要があるとき、NoArk はクラウド上の大規模言語モデルと API を介してやり取りします。そして、それらの提供元の API 利用規約のもとでは、あなたのコンテンツは保存されることも、モデルの学習に使われることもありません。要するに、あなたの記憶はあなたとともにあり、私たちがそれを溜め込むことは決してなく、私たちが呼び出す AI モデルもそれを保持しないのです。そしてもう一つ理由があります。データを溜め込まないことで、上昇し続けるストレージのコストも抑えられるのです——一石二鳥、あなたにとっても、私たちにとっても良いことです。

なぜ NoArk という名前なのですか？

NoArkには二重の意味があります。第一に、ノアの方舟（Noah's Ark）——変化の洪水を、大切なものを乗せて安全に渡るための船——を想起させます。第二に、「no ark（方舟はない）」と読むこともできます。急速に変わるエージェントの時代に、出来合いの方舟であなたを救いに来る人はいません。あなた自身が方舟にならなければならない——NoArkのAIエージェントは、あなたの知識・判断・記憶を前へ運ぶことで、まさにそれを助けます。

なぜ「Infinity Memory」なのですか？

この名前はインフィニティプールから生まれました。シンガポール旅行で、ホテルのインフィニティプール——水面が縁もなく広がっていくように見える——のそばに座っていたとき、もしAIが限界なく記憶できたら、まさにこんな感じだろう、無限の記憶だと気づきました。そこで私たちはNoArkを作り、Charlieに非常に長い記憶を与えました。あなたの文脈・会話・知識を、縁に行き当たることなく前へと運びます。

「Fiat Lux」とはどういう意味ですか？

Fiat Lux はラテン語で 「光あれ」 という意味で、NoArkが生まれたカリフォルニア大学のモットーでもあります。アプリ内では、Fiat Lux は専用のタブです。それは私たちがここで成し遂げたいことを表しています。人々に光をもたらし、そして誰の中にもある光——最良のもの——を引き出すことです。

シャノンのエントロピー（情報量）とは何ですか？

1948年、クロード・シャノンはエントロピーを、情報の流れが持つ不確実性——いわば「驚き」の度合い——として定義しました。メッセージが予測しにくいほど、それが運ぶ情報は多く、「ビット」で測られます。これはデータ圧縮から誤りのない通信まで、デジタル時代の数学的な土台となりました。投資家にとっても直感的です。すでに完全に予測できる価格は何の情報も優位ももたらしません——本当のシグナルは、市場がまだ織り込んでいない部分に宿ります。NoArkのエージェントは、誰もがすでに見えているノイズではなく、その「残された不確実性」を見つけて活かすために作られています。

マクスウェルの悪魔とシャノンのデーモンにはどんな関係がありますか？

どちらも、ランダムさから秩序を生み出すように見える「魔物」です。マクスウェルの悪魔は、物理学者ジェームズ・クラーク・マクスウェルが1867年に思い描いた仮想の存在で、速い分子と遅い分子を選り分けて気体のエントロピーを下げ——熱力学第二法則を破るかに見えます。この逆説が解かれたのは20世紀になってからでした。レオ・シラードが、そして後にロルフ・ランダウアーが、悪魔は選り分けのために情報を取得し最終的に消去しなければならず、その情報の消去自体がエネルギーを要することを示したのです。情報と物理的エントロピーは同じコインの裏表だった——まさにシャノンの情報理論が精密に捉えた結びつきです。彼にちなんで名づけられた「シャノンのデーモン」、すなわちボラティリティを収穫するリバランス戦略は、その金融版のこだまです。変動する価格の無秩序でランダムな揺れから「秩序」——着実な複利——を取り出すのです。両者の深い教訓は同じです。秩序はタダでは手に入りません。マクスウェルの悪魔はエネルギーで支払い、シャノンのデーモンは取引コストで支払う。だからこそNoArkは、秩序を収穫するコストが収穫する秩序を決して上回らない、規律ある低頻度の戦略を好むのです。

「シャノンのデーモン」とは何で、NoArkはそれを使いますか？

テクニカル分析への懐疑に加え、クロード・シャノンは現実の市場の実務的な摩擦も認識していました。彼は、ボラティリティ駆動の高度に体系的なポートフォリオ・リバランス戦略——時に「シャノンのデーモン」と呼ばれる——を概念的に発展させましたが、その数学的アイデアの多くをウォール街で実行することはありませんでした。頻繁な取引の高い手数料と取引コストが、単純なバイ・アンド・ホールド戦略に対する数学的優位をたちまち食いつぶす、と結論づけたからです。NoArkも同じ教訓を胸に刻み、コストが優位を侵食しない、規律ある低頻度の戦略を好みます。

ノーベル賞より前にブラック・ショールズ式を解いた人がいて、LTCMへの投資を断ったというのは本当ですか？

はい——しかも彼はクロード・シャノンの友人であり共同研究者でした。数学者 エドワード・ソープ は、有名な1973年の論文より何年も前に、本質的に ブラック・ショールズ と等価なオプション価格式を導き、それを使って史上最も成功したヘッジファンドの一つを運用しました。転機はこうです。マイロン・ショールズ と ロバート・C・マートン——その式で1997年のノーベル賞を共同受賞——がヘッジファンド ロングターム・キャピタル・マネジメント（LTCM） を共同設立したとき、ソープは投資を断りました。LTCMはその後1998年に劇的に破綻し、ウォール街の救済を必要としました。私たちが胸に刻む教訓はこうです。優雅な数学だけでは足りない——規律あるリスク管理、そしてレバレッジと現実のコストへの敬意こそが、本当に長続きするのです。

リスクの計測にシャープレシオを使いますか？

いいえ——私たちはそれに懐疑的です。ノーベル賞受賞者ウィリアム・シャープが提唱したシャープレシオは、標準偏差1単位あたりのリターンを評価します。しかしそれは 過去を語るだけで、未来を語りません。そして 標準偏差をあたかもリスクと等しいかのように扱います が——私たちはそれを受け入れません。ボラティリティはリスクではありません。本当のリスクは資本の恒久的な損失です。戦略は、破綻する直前まで美しい過去のシャープレシオを示し得ます（LTCMを参照）。だから私たちは、過去を振り返るボラティリティ統計を最適化するのではなく、恒久的な損失を避けることに注力します——決してレバレッジを使わず、ロングオンリーで全額自己資金です。
